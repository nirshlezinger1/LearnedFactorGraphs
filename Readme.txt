This folder contains the source used for evaluating SP over learned factor graphs (StaSPNet). It was tested using Matlab version 2019b with the Deep Learning toolbox.

The main scripts used are:
Test_LOO - sleep pattern detection leave-one-out cross validation train + test
Test_PerPatient - sleep pattern detection per patient train over 1000 samples + test
Test_SymRecovery - Symbol recovery for linear AWGN channel with memory order 4.
